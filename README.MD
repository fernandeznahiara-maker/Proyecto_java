# Planificador de Actividades de Estudio

Proyecto integrador del curso de JavaScript - Manipulación del DOM

## Funcionalidades implementadas

### Obligatorias
- Alta de actividades con validación
- Listado dinámico en tabla usando `<template>`
- Marcar como completada / pendiente
- Eliminar actividades
- Estadísticas actualizadas (total, completadas, pendientes, horas)

### Extras
- Edición de actividades con modal
- Tema claro / oscuro persistente (localStorage)
- Filtro por estado (todas / pendientes / completadas)
- Búsqueda por título o materia
- Ordenamiento por fecha, prioridad o título
- Actividades importantes destacadas
- Mensajes de éxito/error elegantes
- Colores de prioridad (alta roja, media ámbar, baja verde)

## Cómo probarlo

1. Cloná el repositorio
2. Abrí la carpeta con Live Server (o simplemente hacé doble clic en `index.html`)
3. ¡Listo!

## Tecnologías usadas
- HTML5
- CSS3 (variables, flexbox, grid, transiciones)
- JavaScript vanilla (sin frameworks)
